 # ﻿*** Pull Request Reviewer Service ***

## HTTP-сервис для автоматического назначения ревьюверов на Pull Request’ы внутри команды.
Поддерживает создание команд, управление пользователями и автоматическое/ручное назначение ревьюверов на PR.

### Overview

Реализован в соответствии с OpenAPI-спецификацией.
Взаимодействие происходит через HTTP API.
Сервис запускается одной командой ``` docker-compose up --build ```

### Основные возможности

- Автоматическое назначение до 2 активных ревьюверов из команды автора
- Исключение автора из списка кандидатов
- Переназначение ревьювера по правилам доменной модели
- Idempotent Merge (повторные запросы не изменяют состояние)
- Управление активностью пользователей
- Получение списка PR, назначенных конкретному ревьюверу
- Статистика назначений

## Tech Stack

- Go 
- PostgreSQL
- Docker / Docker Compose
- k6 (load testing)
- golangci-lint

## Run Instructions
Запуск приложения и базы данных
``` docker-compose up --build ```

## API доступен по адресу:

http://localhost:8080
